# Scrolls

Personal context modules for AI-assisted conversations.

Published as a GitHub Pages site so that AI voice assistants can reference personal context during conversations where file uploads aren't practical (e.g. Grok in a Tesla, on a walk, or during a drive).

## Files

- `biographical.md` — historical formation
- `philosophical.md` — intellectual positions
- `lifestyle.md` — current life structure
- `dietary.md` — eating, supplements, exercise
- `social.md` — relationships and community
- `financial.md` — financial values framework

## Privacy

Content has been sanitised to remove:
- Names of third parties (family, friends, teachers, colleagues)
- Personal health information about anyone other than the author
- Specific addresses (village/street level)
- Precise financial figures, share counts, account values
- Named references to individuals in institutional criticism

The framework, values, and substance are preserved.
